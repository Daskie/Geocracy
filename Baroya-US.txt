CSC 308-01
NGA - User Stories
17 January 2018


As a beginner player, I want a tutorial level, so that I can learn and understand the game and its controls.

As a user, I want a computer player, so that I can play when I want and where I want, independently.

As a player, I want to be able to rotate the world with a swipe, so that I can have an ease of gameplay and a fluid user experience.
